python中的特殊用法

python 中有一些特殊语法，用来写代码，会使代码更加简洁。

一、filter

filter函数会对给出的序列进行过滤操作
filter函数的定义：
filter(function or None, sequence) -> list tuple or string
function是一个谓词函数，接收一个函数，返回True或False
filter函数会对序列sequence中的每个元素调用function函数，最后返回的结果包括调用结果为True的值。
返回值的类型与sequence的类型一致。
举例（求能被2整除的数）

二、map

map函数会根据提供的函数对指定的序列做映射。
map函数的定义：
map(function, sequence[, sequence, …]) -> list
这个函数的第一参数是一个函数，剩下的参数是一个或多个序列，返回值是一个集合。
function函数可以理解为是一个一对一或多对一的函数，map的作用是以参数序列中的每一个元素调用function函数，返回包含=每次function函数返回值的list。

三、reduce

reduce函数会对参数序列中元素进行累积。
reduce函数定义：
reduce(function, sequence[, initial]) -> value
function是一个有两个参数的函数，reduce会依次在序列sequence中取一个元素，和上一次调用function的结果做参数再次调用function。
第一次调用function时，若有提供initial值，会以sequence中的第一元素和initial作为参数去调用function，否则会以sequence中的前两个元素为参数调用function。
